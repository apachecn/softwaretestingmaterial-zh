## **SQL 变更查询:**

SQL ALTER TABLE 语句用于添加、删除或修改现有表中的列。

**在表格中添加列–语法:**

```
ALTER TABLE table_name ADD column_name datatype;
```

**举例:**

```
ALTER TABLE SCOREBOARD ADD DotBalls int;
```
